Indica la diferencia entre los siguientes métodos, y demuestra su uso con algunos arrays: Array.prototype.forEach(), Array.prototype.every(), Array.prototype.some() y Array.prototype.filter().

Array.prototype.forEach()

Recorre el array y ejecuta la función indicada una vez por cada elemento.

Array.prototype.every()

El método every comprueba si todos los elementos del array cumplen una condición, devolviendo true en caso afirmativo o false en caso negativo.

Array.prototype.some()

Es parecido al every, pero, si tan solo un elemento cumple la condicón devuelve true, y false en caso negative.
